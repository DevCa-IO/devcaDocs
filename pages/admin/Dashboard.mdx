# Dashboard Metrics Overview

![Dashboard Overview](/images/admin-dashboard.png)

The dashboard displays four key metrics related to modem and proxy management:

## Metrics Explained

### Total Modem

Represents the total number of modems in the system. In this case, there are 3 modems registered.

### Total Proxy

Shows the complete count of proxies that have been generated. Currently displaying 14 total proxies.

### Number of Available Proxies

Indicates how many proxies are currently available and ready for use. The system shows 2 proxies are available.

### Modems Paused

Displays the count of modems currently in a paused state. At present, 2 modems are paused.
